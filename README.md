# 基于WebSocket的实时数据传输

## 获取

```shell
git clone -b websocket https://github.com/guzhongren/WebStudy.git
```

## 安装依赖

```shell
npm install # yarn
```

## 启动服务端

```shell
npm run dev
```

## 启动浏览器

* 需提前安装live-server

```shell
npm install -g live-server
```

* 以9000端口为例启动

```shell
live-server --port=9000
```
